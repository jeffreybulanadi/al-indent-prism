# Change Log

All notable changes to the "AL Indent Prism" extension will be documented in this file.

## [1.0.0] - 2025-11-26

### Added
- Initial stable release
- Rainbow indentation colorization for AL language files
- Configurable color palettes for light and dark themes (6 colors each)
- Customizable indent size setting (default: 4 spaces)
- Real-time decoration updates as you type
- Enable/disable toggle setting
- Support for both spaces and tabs
- Automatic theme detection (dark/light)
- Performance-optimized with 100ms debounce
- Zero impact on non-AL files
